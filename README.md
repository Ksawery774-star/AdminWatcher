# AdminWatcher v2.3

**Perfekcyjny plugin do monitorowania administracji** – czysty kod, realne wykrywanie nadużyć, łatwa konfiguracja.

### Co nowego / ulepszone:
- Lepsze parsowanie komendy `/give` (widzi kogo i co dostał)
- Wykrywanie `/gamemode` na **innych graczach**
- Więcej opcji w configu (auto-warn, progi)
- Jeszcze czystszy i bardziej ludzki kod
- Lepsze logi z kontekstem

Plugin gotowy do użycia na produkcyjnym serwerze.
